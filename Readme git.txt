
====================
Setting Git username
====================
git config --global user.name "Mona Lisa"


=====================
Checking Git username
=====================
git config --global user.name


==================================
Adding new files to Git repository
and stages them for commit.
==================================
git add .


================================
Checking the status of the files
in the folder
================================
git status


=============
How to commit
=============
git commit -m "Fixed all the bugs"


=================
Pushing to Github
=================
git push origin master


===================
Pulling from Github
===================
git push



=======================
Atom - getting packages
=======================
Ctrl + Shift + p
Search for "View installed packages"
-------------------
Suggested Packages:
-------------------
  language-batch
    Install this so Atom can recognise windows batch files.
    -------------------
  script
    Install this to run python scripts from within Atom. Use in conjunction with
    language-batch to also run windows batch files. To run something just have
    it open and press "Ctrl + Shift + b".
    -------------------
